# Game Industry Analysis

Аналитический проект по исследованию развития игровой индустрии в начале XXI века.

## О проекте

Цель проекта — проанализировать динамику продаж видеоигр по платформам, жанрам и регионам, а также определить факторы, влияющие на коммерческий успех игр.

В рамках работы выполнены:
- предобработка и очистка данных;
- анализ продаж по годам и платформам;
- исследование популярности жанров;
- анализ региональных различий продаж;
- выявление факторов успешности игр.

##  Данные

Использован датасет с информацией о видеоиграх:

- название игры
- платформа
- год выпуска
- жанр
- продажи по регионам (NA, EU, JP, Other)
- оценки пользователей и критиков
- рейтинг ESRB

##  Инструменты

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

##  Основные результаты

- выявлены платформы-лидеры по количеству выпущенных игр и продажам;
- определены наиболее популярные жанры в разные годы;
- обнаружены различия в предпочтениях игроков по регионам;
- определены факторы, связанные с успешными продажами игр.
